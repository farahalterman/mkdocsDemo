## About
This documentation was created as part of an assignment for the COMM 2216 course at the [British Columbia Institute of 
Technology](https://www.bcit.ca/) (BCIT). The documentation is designed to be at the level of a new student to the Computer Systems Technology 
(CST) program. The focus of this documentation is setting up the [PyCharm IDE](https://www.jetbrains.com/pycharm/) which is used in the COMP 1510 "Programming
Methods" course. 

This documentation was created using [Material for MkDocs](https://squidfunk.github.io/mkdocs-material/) and written in Markdown. User testing and peer review was
utilized to ensure the instructions were easily understandable and accurate.
