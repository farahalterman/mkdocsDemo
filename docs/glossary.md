| Term                   | Definition                                                                                                                |
|------------------------|---------------------------------------------------------------------------------------------------------------------------|
| **Clone**              | Creates a local copy of a GitHub repository                                                                               |
| **GitHub Token**       | A string of characters used to link PyCharm IDE to your GitHub account.                                                   |
| **IDE**                | Integrated Development Environment, a specialized software program designed to allow developers to efficiently write code.|
| **Plugin**             | An add-on that enhances the IDE's default capability in some way.                                                         |
| **Pull**               | Updates local copy to match the GitHub repository                                                                         |
| **Push**               | Updates GitHub repository to match the local copy                                                                         |
| **PyCharm**            | An IDE made by JetBrains that is designed for use with the Python language.                                               |
| **Python**             | A programming language taught in the COMP 1510 course at the Downtown campus (DTC) of BCIT.                               |
| **Repository**         | A place to store files and code                                                                                           |
