| Symptom                                                                      | Possible Cause                                                            | Solution                                                                                                                                                           |
|:-----------------------------------------------------------------------------|---------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Unable to locate the "AI Code Completion" or "AI Inline Completion" setting. | The bundled "AI and Machine Learning" plugins have already been disabled. | Ensure the plugins "Machine Learning Code Completion", "Machine Learning in Search Everywhere", and "Full Line Code Completion" are enabled.                       |
| Your browser does not open when clicking [Log in with JetBrains Account...]. | An unknown error occurred.                                                | Try to log-in manually by clicking the [Log in manually] button on the small PyCharm popup window.                                                                 |
| Your browser does not open when clicking [Generate] to create a new token.   | An unknown error occurred.                                                | Close the popup and retry the step.                                                                                                                                |
| Your JetBrains account login fails.                                          | An unknown external error occurred.                                       | Ensure that you have a [JetBrains account](https://www.jetbrains.com/academy/student-pack/) with the student pack and are signing in with the correct credentials. |
| Only Python versions below 3.10 are available.                               | You have not installed the most recent version of Python.                 | Go to [python.org](https://www.python.org/) and follow the installation instructions.                                                                              |
